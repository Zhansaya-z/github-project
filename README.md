# Simple Interest Calculator

A simple command-line application written in Python that calculates simple interest based on user input.

## Description

This program calculates simple interest using the standard formula:

Simple Interest = (Principal x Rate x Time) / 100

The user enters the principal amount, the annual interest rate, and the time period in years. The program then outputs the calculated interest and the total amount.
